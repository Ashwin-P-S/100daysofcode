## Linked List Implementation of Stack ADT


### Stack

    [element_n]  <- Top
    .
    .
    .
    [element_3]
    [element_2]
    [element_1]

### Operation:

- Push
- Pop
- Top
- Display Stack

### Push:

- INSERT at Top

### Pop:

- REMOVE & RETURNS the Top Element

### Top:

- RETURNS Top Element
